# Introduction

## What's this?

`WKWebViewController` is an UIViewController with WKWebView inside.

## Requirements

* iOS 9.0+
* Xcode 9 with Swift 4

## Installation

#### CocoaPods

```ruby
pod 'WKWebViewController'
```
